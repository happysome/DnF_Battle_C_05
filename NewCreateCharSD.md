```mermaid
sequenceDiagram







actor 사용자 as 플레이어







participant 생성UI as Create\_Character\_UI







participant 전투객체 as 전투







participant 플레이어객체 as 플레이어











사용자->>생성UI: 플레이어id, 캐릭터명, 직업, 레벨 입력







생성UI->>전투객체: 캐릭터생성(플레이어id, 캐릭터명, 직업, 레벨)











전투객체->>플레이어객체: 플레이어체크(플레이어id)











alt 플레이어체크 결과 == true







플레이어객체-->>전투객체: true











alt 직업 == "전사"







%% 전투객체가 전사 객체를 동적으로 '생성'하는 표기법 (박스로 화살표가 꽂힘)







전투객체->>+전사객체: create(캐릭터명, 레벨)







Note right of 전사객체: 내부 로직:<br/>HP = 레벨 × 100<br/>공격력 = 레벨 × 15







전사객체-->>-전투객체: 전사 객체 반환







else 직업 == "마법사"







%% 전투객체가 마법사 객체를 동적으로 '생성'하는 표기법







전투객체->>+마법사객체: create(캐릭터명, 레벨)







Note right of 마법사객체: 내부 로직:<br/>HP = 레벨 × 60<br/>공격력 = 레벨 × 25







마법사객체-->>-전투객체: 마법사 객체 반환







end











%% CD에서 캐릭터생성의 반환타입이 '캐릭터'이므로 생성된 객체를 UI로 반환







전투객체-->>생성UI: 캐릭터 객체 반환







생성UI-->>사용자: 캐릭터 생성 완료 출력











else 플레이어체크 결과 == false







플레이어객체-->>전투객체: false







전투객체-->>생성UI: "플레이어체크 실패"







생성UI-->>사용자: 오류 메시지 출력







end
